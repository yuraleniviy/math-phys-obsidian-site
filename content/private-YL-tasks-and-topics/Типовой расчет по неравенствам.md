## Условия

1. $5x - 7 > 13$

2. $3 - 2x \le 11$

3. $\dfrac{x}{4} + 6 < 1$

4. $7(2 - x) \ge 14$

5. $\dfrac{3x - 5}{2} > \dfrac{x + 7}{4}$

6. $0.6x - 1.2 \le 2.4$

7. $\dfrac{2x + 1}{3} - \dfrac{x - 2}{2} > 1$

8. $4 - (3x - 8) < 2x + 1$

9. $\dfrac{5 - x}{2} \ge \dfrac{x + 1}{3}$

10. $\dfrac{2}{x} > 0,\; x \ne 0$

11. $3(2x - 1) - 5 \le 2(x + 4)$

12. $7x - 3 > 4x + 9$

13. $2(5 - x) \ge 3x - 10$

14. $\dfrac{4x - 1}{5} < \dfrac{2x + 7}{3}$

15. $9 - 4(x - 2) > 3x + 1$

16. $\dfrac{x - 3}{2} + \dfrac{x + 1}{4} \le 5$

17. $0.25(8x - 4) < 3x + 2$

18. $5(1 - 2x) - 3(x - 4) \ge 7$

19. $\dfrac{3x + 2}{6} - \dfrac{x - 1}{4} > \dfrac{1}{3}$

20. $2 - \dfrac{x - 5}{3} \le \dfrac{2x + 1}{2}$

21. $\begin{cases}2x - 3 > 1,\\ x + 4 \le 10\end{cases}$

22. $\begin{cases}5 - x \ge 2,\\ 3x + 1 < 13\end{cases}$

23. $\begin{cases}4x - 7 \le 9,\\ 2x + 5 > 1\end{cases}$

24. $\begin{cases}\dfrac{x - 2}{3} > 1,\\ \dfrac{2x + 1}{5} \le 3\end{cases}$

25. $\begin{cases}7x - 8 < 3x + 12,\\ x - 5 \ge -2\end{cases}$

26. $\begin{cases}3(2x - 1) \le 4x + 5,\\ 2 - x < 0\end{cases}$

27. $\begin{cases}\dfrac{3x - 4}{2} > x + 1,\\ \dfrac{x + 5}{3} \le 4\end{cases}$

28. $\begin{cases}0.5x - 1 \ge 2,\\ 4 - x > -3\end{cases}$

29. $\begin{cases}\dfrac{2x - 3}{4} < \dfrac{x + 1}{2},\\ 5x + 6 \ge 1\end{cases}$

30. $\begin{cases}6 - 2x \le x + 3,\\ \dfrac{x - 1}{2} > -4\end{cases}$
